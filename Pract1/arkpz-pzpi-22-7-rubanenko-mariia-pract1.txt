1 ОСНОВНІ РЕКОМЕНДАЦІЇ НАПИСАННЯ КОДУ НА МОВІ ПРОГРАМУВАННЯ PYTHON

1.1 Мета роботи 

Ознайомитися з ключовими рекомендаціями щодо написання чистого, ефективного та легкого для підтримки коду в різних мовах програмування, а також навчитися аналізувати та рефакторити код для підвищення його якості.

1.2 Хід роботи 

1.	Обрано мову програмування Python для дослідження правил написання коду та проведено аналіз різних джерел.
2.	Підготовлено приклади коду для демонстрації конкретних принципів написання.
3.	Створено презентацію (див. дод. А), у якій висвітлено такі ключові аспекти:
	важливість правил оформлення коду;
	структура коду;
	форматування коду;
	іменування (іменування змінних, функцій, класів);
	коментарі;
	документування коду;
	конвенції стилю кодування для конкретних мов;
	кодування на основі тестування (Test-Driven Development);
	інструменти та автоматизація для підтримки якості коду.

4. Оформлено звіт до практичного заняття.


1.3 Висновок

Під час виконання роботи було детально вивчено основні принципи написання чистого, ефективного та підтримуваного коду на мові Python. Особлива увага була приділена аналізу та рефакторингу існуючого коду, зокрема застосуванню конвенцій стилю, оптимальному іменуванню змінних, функцій та класів, правильному форматуванню коду, а також впровадженню принципів розробки на основі тестування (TDD) та автоматизації процесів для покращення якості коду. Ці підходи допомагають виявляти потенційні проблеми та підвищувати підтримуваність програмних рішень. Розробка презентації дозволила узагальнити отримані знання та підкреслити важливість таких аспектів, як документація коду та використання інструментів для перевірки стилю коду, що є важливими для забезпечення якості програмних продуктів на Python.
